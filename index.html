    Rules

    Welcome to MSSNGVWLS! The goal of the game is to fill in the missing vowels in the words given in the clues. Each clue represents a word with its vowels removed. Type in the correct word in the input box and the box will turn green when you are correct. Complete all the clues as fast as possible! Here are the rules:

    • Each day there is a new category and a set of 5 clues.
    • Each clue represents a word with its vowels removed.
    • Type the correct word in the input box for each clue.
    • The input box will turn green when the correct word is entered.
    • When all words are correct, the timer stops, and your performance is evaluated.
    • If you give up, the correct answers will be revealed.
